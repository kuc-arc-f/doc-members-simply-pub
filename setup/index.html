 インストール手順

  • 2018-10-11

■ インストール手順の流れ

前程： wordpress 4.9.8 がセットアップされている事。
=> 上記version 以外は、動作しない場合があります。

  • 拡張分の、テーブル追加
    => wordpress と、同一ＤＢに追加。

  • インストール
    => 主に、ファイルのコピー

  • config設定
    => ＤＢ接続先等、サブシステム全体の設定

  • /index.php の修正

■ github から、拡張部分のコードを取得。

git clone 又は、ダウンロード

https://github.com/kuc-arc-f/doc-members-simply

=> 設置例は、wordpress を。ROOT/wpuser に
インストールした場合です。

=> ある程度、LAMP系の設定知識がある方を対象にしています。

■ 拡張分の、テーブル追加

=> 認証に、必要なテーブルを追加します。
　ext_users.sql
　https://github.com/kuc-arc-f/doc-members-simply/blob/master/wpuser/ext/db/ext_users.sql

■ インストール

=> ext フォルダー, inedx.php を、wpuser下に
　コピーします。

■ config設定

config.php
https://github.com/kuc-arc-f/doc-members-simply/blob/master/wpuser/ext/data/config.php
に、設定を追加します。
DB host, DB name, DB ユーザー名、DB パスワード

■ /index.php の修正

=>ログイン認証の、処理を追加します。
https://github.com/kuc-arc-f/doc-members-simply/blob/master/wpuser/index.php

■ 管理者機能を使用する場合

  • 管理者ログイン
    => 初回ログインは、下記ユーザーで
    でログイン可能です。

    user: admin@test.com
    pass: 123
    　
    注意点：
    　ログイン後、管理者を追加して。
    　上記の、初回ログイン　ユーザーは、原則削除して下さい。

■ その他

必要あれば、下記機能が利用可能です。
wordpress から、リンクを追加すると。機能を追加できます
=>ユーザー編集
/wpuser/ext/user/?fn=user_edit

=>ユーザー　,ログアウト
/wpuser/ext/user/?fn=user_logout
